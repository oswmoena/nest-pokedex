# Ejecutar en dsarrollo

1. Clonar el repo
2. Ejecutar

```
   npm i
```

3. Tener NestCLI instalado

```
   npm i -g @nestjs/cli
```

4. Levantar BDD

```
   docker-compose up -d
```
